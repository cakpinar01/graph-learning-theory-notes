# Limitations of Finite Pattern Sets in the WL Hierarchy

This note shows that **finite pattern sets cannot match the expressive power of higher-order Weisfeiler–Lehman (WL)**.

## Contents

- `notes.pdf` → proof explanation  
- `demo.ipynb` → small illustration  

## Main Idea

Finite patterns only capture **local structure**, while higher-order WL can detect **global properties** (e.g., cycle structure).  
This creates a separation in expressive power.
